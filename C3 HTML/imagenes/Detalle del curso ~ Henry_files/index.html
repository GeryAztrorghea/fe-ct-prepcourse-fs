Flexbox

Por HENRY ㅤHENRY ㅤ

Flexbox es un valor dentro de CSS destinado a la propiedad display. Permite posicionar, alinear y direccionar los elementos HTML en la pantalla de una manera muy sencilla...

¿Cómo utilizar flexbox?

En el siguiente video aprenderás el paso a paso para utilizar esta propiedad y diseñar interfaces web con poco código, alinear elementos de forma vertical y horizontal, y reordenar contenido sin tocar el HTML.
